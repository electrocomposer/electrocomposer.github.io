API Documentation

electroComposer
Base URL:
  • Tracks: https://ecapi.olk1.com/tracks
    • Returns all track information fields/the entire data set.

How to use

Track Response Fields:
  • id (Integer): Unique identifier for the track.
  • trackName (String): The name of the track.
  • albumName (String): The name of the album.
  • artistName (String): The artist's name.
  • trackNumber (Integer): Track number on the album.
  • trackDuration (Float): Duration of the track (e.g. mm:ss).
  • albumDuration (Float): Duration of an album (e.g. mm:ss).
  • genre (String): The musical genre of the track.
  • artwork (String): The accompanying image for the track. (Album Artwork)
  • ytUrl (String): The URL which links to a YouTube video for the track.
  • ytPlaylistUrl (String): The URL which links to a YouTube playlist for the album, on which the track features.

Query Parameters (where n = a numerical value)

Pagination:

  • /tracks?id=n Get a track by id.
  • /tracks?_limit=n Limits the number of results returned.
  • /tracks?_start=n Defines the starting index for fetching data.
  • /tracks?_start=n&_limit=n Defines the pagination range.

Sorting:

  • /tracks?_order=desc Specifies the sorting order (asc or desc/reverse).

Filtering:

  • /tracks?trackMin=.1&trackMax=.9 Filter by track duration. (seconds)
  • /tracks?albumMin=50&AlbumMax=60 Filter by album duration. (mins)
  • /tracks?genre=New Wave Filter match exact values (e.g. genre=New Wave).

Search:

  • /tracks?_search=heart Full-text search on albumName and trackName.
  • /tracks?trackName=heart Search track name only.
  • /tracks?albumName=heart Search album name only.

Random:

  • /tracks?_random=5 Fetches a random subset of the dataset.

Combine/chain queries parameters together (using &):

  • /tracks?_sort=albumDuration&_order=desc&_limit=5&genre=New Wave
    Fetch tracks by album duration, in descending order, limited to 5 results, in a specific genre.
Base URL:
  • Albums: https://ecapi.olk1.com/albums
    • Returns all album information fields.
Album Response Fields:
  • id (Integer): Unique identifier for the album.
  • albumName (String): The name of the album.
  • artistName (String): The artist's name.
  • albumDuration (Float): Duration of the album (e.g. mm:ss).
  • trackCount (Integer): Number of tracks on the album.
  • genre (String): The musical genre of the album.
  • artwork (String): The accompanying image for the album. (Album Artwork)
  • ytPlaylistUrl (String): The URL which links to a YouTube playlist for the album.

Example Requests:

  • GET https://ecapi.olk1.com/tracks?id=42
  • GET https://ecapi.olk1.com/tracks?_limit=50
  • GET https://ecapi.olk1.com/tracks?_start=200&_limit=50
  • GET https://ecapi.olk1.com/tracks?trackMin=1&trackMax=3&_order=desc&genre=Electronic
  • GET https://ecapi.olk1.com/albums?_sort=albumDuration&_order=asc
  • GET https://ecapi.olk1.com/albums?albumMin=25&albumMax=30&_sort=albumDuration&_order=desc
